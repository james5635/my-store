**Calculator** is a lightweight, cross-platform calculator application built with the Qt Framework using C++. This is a clean, simple utility designed for basic mathematical operations.

### Key Features:
- **Qt Framework Integration** – Modern, professional GUI built with industry-standard Qt
- **Cross-Platform Support** – Runs on Linux and other Qt-supported platforms
- **Simple & Lightweight** – Minimal footprint with fast performance
- **Open Source** – GNU GPL v3.0 license

### Technical Stack:
- **Primary Language:** C++
- **GUI Framework:** Qt Framework (Qt5)
- **Build System:** Custom shell scripts for easy compilation
- **License:** GNU General Public License v3.0
