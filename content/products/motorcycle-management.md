**Motorcycle Shop Management** is a comprehensive mobile application built with Flutter that enables efficient management of a motorcycle shop's operations. The system supports both customer and administrator roles with a full suite of business features.

### Key Features:
- **User Authentication** – Secure login system for customers and administrators
- **Product Management** – Catalog and inventory control for motorcycle parts and accessories
- **Order Management** – Complete order processing and tracking system
- **User Management** – Administrator controls for managing customer and staff accounts
- **Chat System** – Real-time communication between customers and shop staff
- **FAQ System** – Self-service knowledge base for common inquiries

### Technical Stack:
- **Frontend:** Flutter/Dart
- **Backend:** Java with Spring Boot 
- **Native Components:** C++ (8.5%) with CMake build system (6.6%)
- **License:** Apache License 2.0
